Track Titans is a Unity game where trains race, switch lanes, collect power-ups, and avoid obstacles while vying to reach the finish line first!

***Play the game on itch.io***
https://dylspeed100.itch.io/track-titans


Built with:
- Unity (URP)
- C#
- WebGL

Project Structure
- 'Assets/' - Game assets and scripts
- 'Packages/' - Unity packages
- 'ProjectSettings/' - Unity project configuration
